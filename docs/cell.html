Cell¶

class biosim.cell.Cell¶

Cell base class, which refers to a area or a square in a map. From this base class all cell types will be based upon, they will in other words be subclasses of Cell.

eat_carnivore(self)¶

Feeds the carnivores in the current cell and updates their weight by doing so. The prey which is eaten by the predator (carnivore) is deleted from the cell.

The carnivore eats till it has eaten an amnount \(F\), which means that it has eaten himself full of herbivores, i.e. \(\sum w_{herb-eaten} \geq F\)

eat_herbivore(self)¶

Makes all herbivores in the current cell eat and updates their weight by doing so.

The eating method is determined by this:

\(F\) is the animals appetite, and \(f_{ij}\) is the current fodder in the cell.

if \(F\le f_{ij}\)

There is enough fodder in the cell, because F is the apetite, and the herbivore eats till it’s full.

if \(0 < f_{ij} < F\)

the animal eats is what is left of fodder, remaining fodder in the cell will be then set to 0.

if \(f_{ij} = 0\)

the animal receives no food at all, because the current fodder in the cell is equal to 0.

mating(self)¶

Mating function which makes all the animals in current cell have intercourse if ‘determine_birth’ is True and there are more than one animal of the same specie in the cell. Adds a new animal class to cell if the current animal is to give birth, and also updates the weight of the birth-giving animal.

migration(self, neighbour_cells)¶

Moves animals from one cell to another cell, whether the animal is to move is determined by the animal instance with the function ‘determine_to_move’. Which cell it will move to is determined by computing the move probability for each cell, and is chosen by using random choice method with fixed probabilities.

gen_fodder(self)¶

Updates current fodder by this equation:

\[f_{ij} \leftarrow f_{ij} + \alpha \times (f_{\text{Sav-max}} - f_{ij})\]

Where \(f_{ij}\) is the current amount of fodder in the cell, \(\alpha\) is the growth parameter for Savannah subclass, and \(\f_{\text{Sav-max}}\) is the maximum amount of fodder for the Savannah cell.

param = {'alpha': 0.3, 'f_max': 300}¶
